# ✈️ AI 여행 플래너 (Travel Plan Service)

사용자의 여행 성향을 분석하고, 그에 맞는 맞춤형 여행 일정을 생성해주는 서비스입니다. Streamlit과 LangChain (OpenAI GPT-4o)을 활용하여 직관적이고 스마트한 여행 계획을 지원합니다.

## 🌟 주요 기능

### 1️⃣ 여행 성향 질문
6가지 질문을 통해 사용자의 여행 준비 스타일, 이동수단 선호도, 선호하는 동행인, 기대 경험, 하루 일정 패턴, 기록 방식 등을 다각도로 파악합니다.

### 2️⃣ 맞춤형 여행 성향 분석
사용자의 답변을 AI가 종합적으로 분석하여, 아래 6가지 주요 타입 중 하나로 성향을 결정합니다.
* **🌿 힐링형**: 조용하고 여유로운 휴식을 즐기는 타입
* **🚴‍♂️ 액티비티형**: 체험, 스포츠 등 활동적인 것을 즐기는 타입
* **🏛️ 탐방형**: 관광지, 문화유산, 새로운 도시 탐험을 즐기는 타입
* **🍽️ 미식형**: 음식과 맛집 탐방을 최우선으로 즐기는 타입
* **📸 감성형**: 예쁜 카페, 사진, 분위기 중심의 만족을 추구하는 타입
* **⚖️ 균형형**: 새로운 경험과 충분한 휴식의 밸런스를 맞추는 타입

### 3️⃣ 여행 정보 입력
여행을 떠날 구체적인 **지역**과 **여행 기간**(시작일 ~ 종료일)을 입력받아 바탕 데이터를 확정합니다.

### 4️⃣ AI 여행 일정표 자동 생성
분석된 개인의 성향, 선택한 여행지, 체류 기간을 종합하여 **매일(Day-by-Day)**의 세부 맞춤형 일정을 생성합니다.
* **실제 장소 기반**: 허위 정보가 아닌 실존 장소 위주의 동선 추천
* **시간 흐름 고려**: 오전/오후/저녁 단위의 효율적인 연결 동선 계획
* **스트리밍 출력**: AI 생성 결과를 대기 시간 없이 실시간(Stream)으로 표시

## 🚀 시작하기

### 1. 사전 준비사항
* Python 환경 필요
* OpenAI API Key 발급 필요

### 2. 설치 및 실행 방법

저장소 클론 및 패키지 설치
```bash
git clone https://github.com/jinsung99123/travel_plan_service.git
cd travel_plan_service
pip install -r requirements.txt
```

API 키 설정
> 앱을 실행하기 위해서 `.streamlit/secrets.toml` 파일에 API 키를 설정해야 합니다. (또는 `.env.example` 포맷을 참고하세요)
```toml
# .streamlit/secrets.toml 파일 내
OPENAI_API_KEY = "발급받은_당신의_API_키"
```

애플리케이션 실행
```bash
streamlit run app.py
```
